# App Privacy Answers Draft

Use this for App Store Connect privacy questions if the app remains as currently designed.

## Data Collection

Data Collected: No

The app does not collect personal data, usage data, diagnostics, contact information, identifiers, location, purchases, search history, browsing history, or user content.

## Tracking

Tracking: No

The app does not track users across apps or websites owned by other companies. It does not include advertising SDKs or analytics SDKs.

## Accounts

Accounts: No

The app does not provide account creation or sign-in.

## Network

The first version should remain offline-first. If future versions add online content, analytics, cloud sync, AI features, or sharing, update this file, the privacy policy, and App Store Connect privacy answers before release.
